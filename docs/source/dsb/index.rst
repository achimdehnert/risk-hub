DSB — Datenschutz Module
========================

The DSB (Datenschutz/Data Protection) module manages VVT (Verarbeitungsverzeichnis),
TOM (Technische und Organisatorische Maßnahmen), and AVV (Auftragsverarbeitungsverträge)
records for multi-tenant occupational safety management.
